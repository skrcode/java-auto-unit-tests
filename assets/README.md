# JAIPilot Assets

This directory contains visual assets for the JAIPilot project.

## Directory Structure

```
assets/
├── screenshots/     # IDE screenshots showing JAIPilot in action
├── demos/          # Animated GIFs demonstrating features
└── logos/          # JAIPilot branding and logos
```

## Creating Demo GIFs

Recommended tools:
- **macOS**: Kap (https://getkap.co/)
- **Windows**: ScreenToGif (https://www.screentogif.com/)
- **Linux**: Peek (https://github.com/phw/peek)

Keep GIFs:
- Under 30 seconds
- Under 5MB after optimization
- At 10-15 FPS
- Showing only relevant screen area

## Image Guidelines

- **Format**: PNG for screenshots, GIF for animations, SVG for logos
- **Resolution**: Minimum 1280x720 for screenshots
- **Optimization**: Always optimize before committing
- **Privacy**: Don't include personal information or API keys
